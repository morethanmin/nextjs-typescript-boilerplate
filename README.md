# nextjs-typescript-boilerplate

회원가입, 로그인, 로그아웃을 구현하였습니다.

## Tech and libraries

### client
- React
- Next.js
- @reduxjs/toolkit
- @emotion (css)
- axios

### server

- express
- MySQL
- sequelize
